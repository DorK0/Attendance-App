# attendance app

attendance-app is a app for tracking users attendance made with 
##### Express & React.

## Installation

```bash
cd backend 
```
```bash
backend: npm i 
```
```bash
cd frontend 
```
```bash
frontend: npm i 
```
```bash
backend: npm start 
```
```bash
frontend: npm start 
```
